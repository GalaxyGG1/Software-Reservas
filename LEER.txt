----------------------------------------------------------------
  DESCRIPCION DEL PROYECTO
----------------------------------------------------------------

Sistema de escritorio desarrollado en C# (Windows Forms) con base
de datos SQL Server. Permite gestionar reservas hoteleras con
control de acceso por roles, modulos de clientes, habitaciones,
facturas, servicios adicionales, reportes y administracion.

----------------------------------------------------------------
  REQUISITOS DEL SISTEMA
----------------------------------------------------------------

  - Windows 10 / 11
  - Visual Studio 2019 o superior (con soporte para .NET Framework 4.8)
  - SQL Server (cualquier edicion, incluyendo Express)
  - SQL Server Management Studio (SSMS)

----------------------------------------------------------------
  PASOS PARA EJECUTAR EL PROYECTO
----------------------------------------------------------------

  1. Abrir SQL Server Management Studio (SSMS).

  2. Ejecutar UNICAMENTE el script:

        Base de Datos\HotelDB_Proyecto_Final.sql

     Este script crea la base de datos completa, todas las tablas,
     indices, funciones, procedimientos almacenados, triggers,
     vistas y los datos de prueba. No requiere ningun script
     adicional.

     (Opcional) Si quieres mas datos de prueba, puedes ejecutar
     ademas:

        Base de Datos\HotelDB_SeedData.sql

  3. Verificar el nombre de tu servidor SQL Server.
     (Se puede ver en la pantalla de conexion de SSMS)

  4. Abrir la solucion en Visual Studio:

        HotelReservas.slnx

  5. Abrir el archivo:

        HotelReservas\Database.cs

  6. Buscar la cadena de conexion (linea 9-10) y cambiar el
     nombre del servidor por el tuyo:

        Antes:  Server=LAPTOP-B1972UH7;
        Despues: Server=TU_NOMBRE_DE_SERVIDOR;

     Ejemplo completo:
        @"Server=MI-PC\SQLEXPRESS;Database=HotelDB;Trusted_Connection=True;"

  7. Guardar los cambios y ejecutar el proyecto (F5).

----------------------------------------------------------------
  TEMAS VISUALES
----------------------------------------------------------------

  El sistema soporta MODO OSCURO (predeterminado) y MODO CLARO.

  MODO OSCURO:
    - Fondo:           Negro profundo  (#0D1117)
    - Panel lateral:   Azul marino     (#0F2547)
    - Botones de menu: Azul marino     (#1A3A6B)
    - Acento / CTA:    Amarillo        (#F5C518)
    - Texto:           Blanco          (#FFFFFF)
    - Barra de estado: Azul muy oscuro (#060E1F)

  MODO CLARO:
    - Fondo:           Blanco          (#FFFFFF)
    - Panel lateral:   Azul claro      (#DDEEFF)
    - Botones de menu: Azul cielo      (#5BA3D9)
    - Acento / CTA:    Naranja         (#FF7A2F)
    - Texto:           Azul marino     (#0F2547)
    - Barra de estado: Azul marino     (#0F2547)

  COMO CAMBIAR EL TEMA:
    - En la pantalla de login:  boton "Modo Claro / Modo Oscuro"
                                (esquina superior derecha)
    - En el panel principal:    boton "Modo Claro / Modo Oscuro"
                                (parte inferior del menu lateral)
    - El tema se aplica al instante sin necesidad de reiniciar.
